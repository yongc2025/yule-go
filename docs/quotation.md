# 渔乐（yule）小程序系统 — 商业报价单

---

## 一、项目概述

| 项目 | 内容 |
|:---|:---|
| 项目名称 | 渔乐 — 渔具店周末短途旅行数字化运营系统 |
| 委托方 | （客户名称） |
| 服务方 | （公司/团队名称） |
| 报价日期 | 2026 年 5 月 5 日 |
| 报价有效期 | 30 个自然日 |
| 文档版本 | V1.0 |

**项目简介：** 为线下渔具实体店打造"零售 + 短途旅行"双盈利模式的数字化运营系统，包含微信小程序（客户端）和 PC 端管理后台，覆盖在线预约、会员充值、装备租赁、老带新裂变、财务统计等核心业务功能。

## 二、项目范围

### 2.1 交付物清单

| 序号 | 交付物 | 说明 |
|:---:|:---|:---|
| 1 | 微信小程序 | 面向 C 端用户，支持在线预约、支付、会员、裂变等 |
| 2 | PC 端管理后台 | 面向店主/运营，支持团期/订单/客户/财务管理 |
| 3 | 后端 API 服务 | Go 语言 RESTful API，支撑小程序和管理后台 |
| 4 | 数据库设计 | MySQL 数据库，含完整的表结构和初始数据 |
| 5 | 部署文档 | 服务器部署指南、运维手册 |
| 6 | 源代码 | 完整源代码，含注释和 README |
| 7 | 测试报告 | 功能测试、接口测试报告 |

### 2.2 功能模块明细

#### 模块一：团期管理（后台 + 小程序）

| 功能点 | 说明 |
|:---|:---|
| 创建团期 | 选择线路、出行日期、最大名额、领队信息 |
| 编辑团期 | 修改名额、领队等信息 |
| 取消团期 | 取消已创建的团期，已报名客户自动通知 |
| 团期列表（小程序） | 按周展示可报名团期，显示剩余名额 |
| 团期详情（小程序） | 展示行程安排、价格、包含项目 |

#### 模块二：用户预约（小程序）

| 功能点 | 说明 |
|:---|:---|
| 线路选择 | 四大线路（垂钓/露营/亲子/长者）详情展示 |
| 人数选择 | 成人/儿童人数，自动计算费用 |
| 联系信息 | 填写联系人姓名、电话 |
| 订单确认 | 费用明细（团费+租赁费-折扣-余额） |
| 微信支付 | 小程序原生支付 |
| 订单列表 | 我的订单，按状态筛选 |
| 订单详情 | 完整订单信息、出行提醒 |
| 取消订单 | 出行前 24 小时免费取消 |

#### 模块三：会员充值（小程序）

| 功能点 | 说明 |
|:---|:---|
| 充值方案展示 | 三档充值（200/500/1000），权益说明 |
| 充值支付 | 微信支付充值 |
| 会员等级 | 自动升级（银卡/金卡/钻石） |
| 余额查询 | 账户余额、充值记录 |
| 自动折扣 | 下单时自动计算会员折扣和旅行立减 |
| 余额抵扣 | 可选择用余额支付订单 |

#### 模块四：装备租赁（小程序 + 后台）

| 功能点 | 说明 |
|:---|:---|
| 装备展示 | 可租赁装备列表（鱼竿/帐篷/天幕/桌椅） |
| 租赁选择 | 下单时勾选装备和数量 |
| 费用合并 | 租赁费随订单一起支付 |
| 库存管理（后台） | 增删改查装备项、调整库存 |
| 库存校验 | 库存不足时提示"已借完" |

#### 模块五：老带新裂变（小程序）

| 功能点 | 说明 |
|:---|:---|
| 邀请码 | 每个用户唯一邀请码 |
| 分享功能 | 生成邀请海报/链接，分享到微信 |
| 邀请关系绑定 | 新用户通过邀请链接注册自动绑定 |
| 首单立减 | 被邀请人首单自动减 15 元 |
| 邀请奖励 | 被邀请人下单后，邀请人返现 20 元到余额 |
| 邀请统计 | 邀请人数、累计奖励查看 |

#### 模块六：管理后台（PC 端）

| 功能点 | 说明 |
|:---|:---|
| 管理员登录 | 账号密码登录，角色权限控制 |
| 首页看板 | 今日/本周/本月营收概览、订单统计 |
| 团期管理 | 创建/编辑/取消团期 |
| 订单管理 | 按状态筛选、查看详情、导出名单、退款操作 |
| 客户管理 | 客户列表、会员等级、充值记录、消费记录 |
| 线路管理 | 编辑四大线路详情、价格、行程 |
| 装备管理 | 增删改查装备项、库存调整 |
| 财务统计 | 按日/周/月营收、按线路利润统计 |

## 三、技术方案

| 维度 | 方案 |
|:---|:---|
| 小程序前端 | uni-app（Vue3 + TypeScript） |
| 管理后台 | Vue3 + Element Plus + TypeScript |
| 后端 API | Go（Gin 框架） |
| 数据库 | MySQL 8.0 |
| 缓存 | Redis |
| 支付 | 微信支付 V3 |
| 文件存储 | 阿里云 OSS / 腾讯云 COS |
| 部署 | Docker + Docker Compose |
