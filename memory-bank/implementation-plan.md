## 执行原则

1. **每个 Step 完成后必须：** 更新 `memory-bank/progress.md` + `tasks/000X/STATUS.md`
2. **每个 Phase 完成后必须：** 更新 `memory-bank/architecture.md`
3. **失败时：** 记录失败原因到 STATUS.md，回退到当前 Phase 的 Step 1 重试
4. **连续 3 次失败：** 标记为 🔴 致命错误，暂停并等待人工介入
