# 需求规格书 — yule-go

> 来源：《渔乐项目商业计划书》结构化提炼
> 本文档 = memory-bank 的「游戏设计文档」，是 AI 理解项目需求的唯一真相源

---

## 1. 项目背景

线下渔具门店经营单一，仅靠渔具零售盈利，同质化严重、利润微薄、客户粘性低。
城市居民周末微度假需求爆发，本地缺少钓鱼露营专属组团服务。

**核心机会：** 依托现有渔具实体店，打造"零售 + 短途旅行"双盈利模式。

## 2. 目标客群

| 客群 | 年龄 | 特征 | 核心需求 |
|:---|:---|:---|:---|
| 垂钓爱好者 | 25-60 岁 | 男性为主，周末时间充裕 | 省心垂钓体验，付费意愿强 |
| 亲子家庭 | 28-45 岁 | 携带 5-14 岁儿童 | 安全、亲子互动、自然体验 |
| 退休中老年 | 55+ 岁 | 时间自由，偏好慢节奏 | 同龄人结伴、低成本、休闲出行 |

## 3. 核心产品（MVP 范围）

### 3.1 四大出行产品

| 产品 | 价格 | 包含项目 |
|:---|:---|:---|
| 成人钓友单日垂钓团 | 128 元/人 | 往返车费、专属钓点、基础饵料、领队服务、饮用水、出行保险 |
| 垂钓+露营一日套餐 | 198 元/人 | 交通、钓点、露营天幕桌椅、基础渔具、简餐、保险 |
| 亲子出游套餐 | 238 元/一大一小 | 交通、儿童趣味钓鱼、亲子游戏、露营野餐、全程陪同、保险 |
| 退休专属慢游单日团 | 98 元/人 | 往返接送、休闲观光、轻度垂钓、农家清淡午餐、全程陪同、保险 |

### 3.2 装备租赁

| 装备 | 价格 |
|:---|:---|
| 鱼竿套装 | 30 元/套 |
| 帐篷 | 50 元/顶 |
| 天幕 | 40 元/个 |
| 折叠桌椅 | 25 元/套 |

### 3.3 会员充值体系（锁客核心）

| 档位 | 渔具折扣 | 旅行立减 | 赠送权益 |
|:---|:---|:---|:---|
| 200 元 | 9.5 折 | 每次立减 10 元 | 钓鱼团优惠券 1 张 |
| 500 元 | 9 折 | 每次立减 20 元 | 装备免费租赁 3 次 |
| 1000 元（主推） | 8.5 折 | 每次立减 30 元 | 免费单日出行名额 1 个，旺季优先留位 |

### 3.4 老带新裂变

- 老客户返现/下次参团立减 20 元
- 新客户首单立减 15 元
- 2 人同行每人立减 10 元
- 3 人及以上每人立减 15 元 + 赠送鱼饵礼包

## 4. 功能需求（MVP）

### 4.1 小程序端（客户）

| 模块 | 功能 | 优先级 |
|:---|:---|:---|
| 首页 | 本周团期展示、热门线路推荐 | P0 |
| 线路详情 | 四大产品详情、行程安排、价格 | P0 |
| 在线预约 | 选日期 → 选线路 → 填人数 → 微信支付 → 生成订单 | P0 |
| 日历排期 | 可视化查看每周六/日团期状态 | P0 |
| 会员中心 | 充值、余额查询、折扣自动计算 | P0 |
| 我的订单 | 历史订单、待出行、已出行、退款 | P0 |
| 装备租赁 | 下单时勾选租赁项，费用合并支付 | P0 |
| 老带新 | 分享邀请码，好友首单立减，自己返现 | P0 |
| 出行须知 | 免责声明、天气改期政策、保险说明 | P1 |

### 4.2 管理后台（店主/运营）

| 模块 | 功能 | 优先级 |
|:---|:---|:---|
| 团期管理 | 创建/编辑每周团期 | P0 |
| 订单管理 | 查看所有订单、名单导出、退款处理 | P0 |
| 客户管理 | 客户列表、会员等级、充值记录 | P1 |
| 线路管理 | 四大线路详情编辑 | P1 |
| 财务统计 | 日/周/月营收、各线路利润 | P1 |
| 装备管理 | 租赁项管理、库存 | P1 |

## 5. 非功能需求

| 维度 | 要求 |
|:---|:---|
| 性能 | API 响应时间 < 500ms，支持 100 并发 |
| 可用性 | 小程序操作不超过 3 步完成预约 |
| 安全 | 微信支付 HTTPS，用户数据加密存储 |
| 兼容性 | 微信小程序基础库 2.20+ |
| 可维护性 | 代码注释覆盖率 > 30%，关键接口有文档 |

## 6. 非目标（MVP 不做）

- ❌ APP 原生开发（先做小程序）
- ❌ 多城市连锁支持（先做单店）
- ❌ 评价/相册系统（V2 再做）
- ❌ 领队排班管理（线下先做）
- ❌ 数据看板（先有数据再说）
- ❌ 短信/推送通知（微信消息先做）
