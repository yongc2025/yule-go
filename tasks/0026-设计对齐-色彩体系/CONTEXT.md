# 0026 — 设计对齐：全局色彩体系

## 用户故事

作为用户，打开小程序后看到的设计风格应该与设计稿一致（森林绿+暖橙，自然温暖），而不是通用的蓝色模板。

## 背景

设计稿 `docs/design/` 定义了完整的色彩体系：
- 主色：森林绿 (#2D6A4F) — 自然、信任
- 强调色：暖橙 (#E76F51) — 活力、行动
- 背景：暖米色 (#F5F5F0)

实际实现使用了蓝色 (#1890ff) + 红色 (#f5222d) + 灰色 (#f5f5f5)。

## 验收标准

- [ ] 全局主色为 #2D6A4F（森林绿）
- [ ] 价格/强调色为 #E76F51（暖橙）
- [ ] 页面背景为 #F5F5F0（暖米色）
- [ ] 导航栏背景为 #2D6A4F
- [ ] 所有按钮、标签、选中态使用新色彩
- [ ] 无残留的蓝色 (#1890ff) 元素

## 约束

- 仅修改 cloud-mvp/miniprogram 下的样式文件
- 不改变功能逻辑
