# CONTEXT — 0006 管理后台

## 需求摘要

PC 端管理后台，提供团期管理、订单管理、客户管理、线路管理、装备管理、财务统计六大模块。

## 验收标准

- AC-01：管理员登录后进入后台首页，显示今日/本周/本月营收概览
- AC-02：团期管理支持创建、编辑、取消团期
- AC-03：订单管理支持按状态筛选、查看详情、导出名单、退款操作
- AC-04：客户管理支持查看客户列表、会员等级、充值记录、消费记录
- AC-05：线路管理支持编辑四大线路的详情、价格、行程
- AC-06：财务统计支持按日/周/月查看营收，按线路查看利润
- AC-07：装备管理支持增删改查装备项、调整库存

## 依赖

- 所有后端 API 模块
- 认证：管理员登录 + JWT
