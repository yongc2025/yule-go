# 0038 用户个人信息

## 用户故事
作为用户，我希望能编辑和展示我的个人资料，包括昵称、头像、手机号、个人简介和相册。这些信息也为 v2 领队人设做铺垫。

## 验收标准
- AC-01: 用户可编辑昵称、头像
- AC-02: 用户可绑定/修改手机号
- AC-03: 用户可填写个人简介
- AC-04: 用户可上传个人相册照片（最多 9 张）
- AC-05: 「我的」页面展示完整个人信息
- AC-06: 信息存储在云端（users 集合），非仅本地缓存

## 数据模型（users 集合扩展）
| 字段 | 类型 | 说明 |
|---|---|---|
| nickname | string | 昵称 |
| avatar | string | 头像 URL |
| phone | string | 手机号 |
| bio | string | 个人简介 |
| album | array[string] | 相册照片 URL 列表 |

## 约束
- 头像和相册图片上传到云存储
- 手机号可通过微信 getPhoneNumber 获取
- 为 v2 领队功能预留 isGuide 字段（暂不启用）
