# Task 0016 — 管理后台补全

## 用户故事
作为店主/运营，我需要完整的管理后台来管理线路、查看财务、管理客户。

## 验收标准
- [ ] AC-01: Dashboard 首页 — 今日数据（订单数/收入/新增用户）、本周趋势图
- [ ] AC-02: 线路管理 — CRUD（创建/编辑/上下架/关联钓场）
- [ ] AC-03: 订单管理增强 — 订单详情页、退款操作、导出 Excel
- [ ] AC-04: 客户管理增强 — 客户详情（订单历史/充值记录/余额变动）
- [ ] AC-05: 财务统计 — 日/周/月报表、按线路/钓场维度统计
- [ ] AC-06: 装备管理增强 — 库存管理、上下架
- [ ] AC-07: 钓场管理页面（依赖 0013）
- [ ] AC-08: 核销页面（依赖 0014）

## 技术要点
- Dashboard 数据需要统计接口（或缓存）
- Excel 导出可用 excelize 库
- 图片上传需要 OSS 或本地存储
