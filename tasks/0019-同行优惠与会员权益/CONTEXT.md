# Task 0019 — 同行优惠与会员权益

## 用户故事
作为用户，我和朋友一起报名能享受同行优惠；作为会员，充值后能享受赠送权益。

## 验收标准
- [ ] AC-01: 2人同行每人立减 10 元
- [ ] AC-02: 3人及以上每人立减 15 元 + 赠送鱼饵礼包
- [ ] AC-03: 同行优惠与会员折扣叠加规则（取最优 or 叠加，需明确）
- [ ] AC-04: 会员充值赠送权益：200 元送优惠券 1 张
- [ ] AC-05: 会员充值赠送权益：500 元送免费租赁 3 次
- [ ] AC-06: 会员充值赠送权益：1000 元送免费出行名额 1 个
- [ ] AC-07: 优惠券系统（创建/发放/使用/过期）
- [ ] AC-08: 免费租赁次数管理（发放/使用/剩余查询）
- [ ] AC-09: 免费出行名额管理（发放/使用）

## 技术要点
- 同行关系判定：同一团期 + 同一批下单 or 关联邀请码
- 优惠券需要独立表（coupons）
- 优惠叠加规则需产品决策
- 免费租赁次数存用户表或独立表
