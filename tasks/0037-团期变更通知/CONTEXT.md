# 0037 团期变更 + 通知用户

## 用户故事
作为店主，因天气等原因需要变更或取消团期时，系统自动通知已报名用户，用户可选择接受变更或申请退款。

## 验收标准
- AC-01: 店主可修改团期日期/名额/集合地点
- AC-02: 店主可取消团期（自动触发全额退款 + 通知）
- AC-03: 团期变更后自动发送微信订阅消息给已报名用户
- AC-04: 用户收到通知后可查看变更详情
- AC-05: 团期取消时自动退款 + 通知

## 通知场景
| 场景 | 通知内容 | 退款 |
|---|---|---|
| 日期变更 | "XX活动日期变更至X月X日" | 不自动退，用户可选 |
| 地点变更 | "XX活动集合地点变更" | 不退 |
| 团期取消 | "XX活动已取消" | 自动全额退 |

## 约束
- 微信订阅消息需用户授权（下单时引导授权）
- 订阅消息模板需在微信公众平台申请
