# Task 0016 — 订单状态机 + 退款

## 用户故事
系统需要完整的订单状态流转，支持退款审核和自动完成。

## 验收标准
- [ ] AC-01: 订单状态扩展为 7 个（PENDING/PAID/CANCELLED/REFUNDING/REFUNDED/EN_ROUTE/COMPLETED）
- [ ] AC-02: 退款规则：72h+全退、24-72h半退、24h内不退
- [ ] AC-03: 管理员退款审核（通过/驳回）
- [ ] AC-04: 退款后释放名额 + 退还余额
- [ ] AC-05: 团期次日自动完成
- [ ] AC-06: 状态机校验（非法转换拒绝）

## 依赖：0014-微信支付, 0012-钓场管理
