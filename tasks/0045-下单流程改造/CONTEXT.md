# 0045 下单流程改造

> **分支**：cloud-mvp
> **设计文档**：`docs/growth-design.md` §4.5, §4.6

## 用户故事
作为用户，下单时我希望看到所有可用的优惠（会员折扣、优惠券、余额抵扣、邀请立减、同行优惠），并自动帮我算出最划算的价格。

## 验收标准
- AC-01: 下单页展示可用优惠券列表，支持选择
- AC-02: 下单页支持余额抵扣开关
- AC-03: 费用明细分项展示（原价/折扣/券/余额/实付）
- AC-04: 被邀请人首单自动立减 ¥15
- AC-05: 3 人+同行自动优惠
- AC-06: 优惠叠加规则正确（折扣+券可叠加，同行取最优惠）
- AC-07: 订单详情展示完整优惠明细

## 优惠叠加规则
```
基础费用 = 成人价×成人数 + 儿童价×儿童数
├── 会员折扣（如有）
├── 邀请立减（被邀请人首单）
├── 同行优惠（3人+，与会员折扣取最优惠）
├── 优惠券（用户选 1 张，可与折扣叠加）
├── 余额抵扣（可与任何优惠叠加）
└── 实付金额 = max(0, 折后价 - 券 - 余额)
```
