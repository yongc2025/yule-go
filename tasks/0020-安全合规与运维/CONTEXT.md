# Task 0020 — 安全合规与运维

## 用户故事
作为系统管理员，我需要确保系统安全合规、可运维。

## 验收标准
- [ ] AC-01: 微信支付 V3 RSA 签名（替换当前 SHA256 简单签名）
- [ ] AC-02: 敏感数据加密存储（手机号 AES 加密）
- [ ] AC-03: 接口频率限制（防刷）
- [ ] AC-04: 管理员操作审计日志
- [ ] AC-05: 小程序隐私协议弹窗
- [ ] AC-06: 用户数据导出接口
- [ ] AC-07: 用户注销功能
- [ ] AC-08: 数据库备份策略
- [ ] AC-09: 服务健康检查接口
- [ ] AC-10: 日志收集方案

## 技术要点
- 微信支付 V3 需要 RSA 私钥 + 平台证书
- 手机号加密用 AES-256-GCM，密钥从环境变量读取
- 频率限制可用 Redis 令牌桶
- 审计日志写入独立表或文件
