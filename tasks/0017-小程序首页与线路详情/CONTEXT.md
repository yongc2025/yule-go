# Task 0017 — 小程序首页与线路详情

## 用户故事
作为用户，打开小程序能看到附近钓场和本周团期，点击线路查看详情。

## 验收标准
- [ ] AC-01: 首页 — 附近钓场（需位置授权，按距离排序）
- [ ] AC-02: 首页 — 本周团期列表
- [ ] AC-03: 首页 — 热门线路推荐
- [ ] AC-04: 首页 — 搜索钓场/线路
- [ ] AC-05: 线路详情页 — 行程安排、价格、包含项目、关联钓场
- [ ] AC-06: 线路详情页 — 关联团期列表（可直接跳转预约）
- [ ] AC-07: 钓场详情页 — 地址、设施、鱼种、图片、关联团期

## 技术要点
- 位置授权需在 app.json 声明
- 附近钓场接口：GET /api/v1/spots/nearby?lat=&lng=&radius=
- 首页数据需缓存优化

## 依赖
- 0013-钓场钓位体系（钓场 API）
