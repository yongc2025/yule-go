## 设计说明

- **主色调**：深蓝 `#1565c0`（水/信任）+ 绿色 `#2e7d32`（自然/户外）
- **点缀色**：金色 `#ffca28`（阳光/活力）+ 橙红 `#ff5722`（渔漂）
- **核心元素**：太阳、山水、水波、鱼钩、渔漂、跃起的鱼
- **字体**：PingFang SC / Microsoft YaHei
