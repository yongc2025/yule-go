# 渔乐（yule-go）

以渔具店为载体的周末短途旅行数字化运营系统。

## 项目概述

- **业务：** 渔具零售 + 周末组团短途旅行
- **客群：** 垂钓爱好者、亲子家庭、退休中老年
- **技术：** Go + Vue3 + uni-app + MySQL + Redis

## 功能模块

| 模块 | 说明 | 状态 |
|:---|:---|:---|
| 团期管理 | 后台创建/编辑团期，前端展示 | 🟡 待开发 |
| 用户预约 | 选线路→填信息→下单支付 | 🟡 待开发 |
| 会员充值 | 三档充值、余额抵扣、折扣 | 🟡 待开发 |
| 装备租赁 | 下单时勾选租赁项 | 🟡 待开发 |
| 老带新裂变 | 邀请码、新客立减、老客返现 | 🟡 待开发 |
| 管理后台 | 团期/订单/客户/财务统一管理 | 🟡 待开发 |

## 快速开始

> ⚠️ **首次开发请先阅读：[本地开发环境搭建指南](docs/local-setup.md)**（含 Windows 踩坑记录）

```bash
# 查看帮助
make help

# 编译后端
make build

# 运行
make run

# 测试
make test
```

## 目录结构

```
yule-go/
├── AGENTS.md           # Agent 行为准则
├── docs/               # 文档（需求、架构、数据库、API）
├── tasks/              # 任务管理
├── workflow/           # 自动化工作流
├── memory/             # 踩坑记录
├── src/                # 源码
│   ├── server/         # Go 后端
│   ├── web/            # 管理后台
│   └── miniprogram/    # 小程序
├── migrations/         # 数据库迁移
├── scripts/            # 脚本工具
└── docker/             # Docker 配置
```

## 开发工作流

采用 Vibe Coding 工作流（详见 `workflow/`）：

1. 需求输入 → 规格锁定
2. 执行计划 → 计划编排
3. 实施变更 → 代码生成
4. 验证发布 → 测试验证
5. 总控循环 → 失败回跳/成功归档
