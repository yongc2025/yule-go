# crypto-quant-framework

学习项目

企业级加密货币量化回测框架 | 事件驱动引擎、多技术因子计算、网格/跨所套利策略、内置风控模块，支持MySQL+Redis数据存储 
Enterprise Crypto Quant Backtest Framework | Event-driven engine, multi-factor calculation, grid/arbitrage strategy, risk management, MySQL+Redis storage


# 通用加密货币量化回测框架V1.0
## 项目功能
1. 自研事件驱动回测引擎（支持Tick/K线级别重放）
2. 集成10+技术因子（RSI/EMA/MACD/VWAP/布林带）
3. 实现网格策略、跨交易所套利策略
4. 仓位管理、止损止盈等风控模块
5. MySQL存储历史数据，Redis缓存实时行情

## 技术栈
Python、Pandas、NumPy、MySQL、Redis、币安/OKX API
## 匹配岗位
量化开发工程师、量化策略研究员、做市策略助理
